This example is for testing BackgroundBlurEffect dither noise strength and blur stability while background content is static or animated.

Controls:

- 1: Toggle blur pane visibility
- 2: Change image board movement speed
- 3: Decrease blur radius
- 4: Increase blur radius
- 5: Change window background color
- 6: Toggle image board animation
- 7: Decrease dither noise strength
- 8: Increase dither noise strength
- Esc/Back: Exit
